Responsividade e Mobile First

7) O que é responsividade?

Responsividade é a capacidade de um site se adaptar a diferentes tamanhos de tela, como celulares, tablets, notebooks e monitores grandes.

Como funciona?

Usamos media queries no CSS para aplicar estilos diferentes dependendo do tamanho da tela.

@media (max-width: 768px) {
  .menu {
    flex-direction: column;
  }
}

Dispositivos comuns

Tamanhos de tela usados como referência:

/* Celular */
max-width: 480px

/* Tablet */
max-width: 768px

/* Notebook */
max-width: 1024px

/* Monitor */
min-width: 1280px

width: 100%

Uma forma simples de responsividade é usar width: 100%, fazendo o elemento ocupar toda a largura disponível.

img {
  width: 100%;
}

.caixa {
  width: 100%;
}

11) Mobile First — o que é e por que usar?

❌ Desktop First

Começa pelo desktop e usa max-width para adaptar ao celular. O celular vira caso especial.

.grid {
  grid-template-columns: repeat(3, 1fr);
}

@media (max-width: 768px) {
  .grid {
    grid-template-columns: 1fr;
  }
}

✅ Mobile First

Começa pelo celular e usa min-width para expandir em telas maiores. É a abordagem recomendada.

.grid {
  grid-template-columns: 1fr;
}

@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
