HTML e Acessibilidade

O que é semântica e sua importância?

HTML semântico significa usar as tags corretas para cada tipo de conteúdo. Isso ajuda navegadores, leitores de tela e ferramentas de busca a entenderem melhor a estrutura da página, tornando o site mais acessível e organizado.

1) O que é semântica?

Semântica é usar a tag certa para o conteúdo certo. Em vez de usar apenas <div> para tudo, usamos tags que descrevem o que o conteúdo representa.

<!-- Sem semântica -->
<div>Menu</div>

<!-- Com semântica -->
<nav>Menu</nav>

2) Principais tags semânticas

Tags que descrevem a estrutura da página:

<header>   — cabeçalho
<nav>      — navegação
<main>     — conteúdo principal
<section>  — seção temática
<article>  — conteúdo independente
<aside>    — conteúdo lateral
<footer>   — rodapé

3) O que é acessibilidade?

Acessibilidade é desenvolver sites que possam ser usados por todas as pessoas, incluindo usuários com deficiência visual, motora, auditiva ou cognitiva.

4) Como melhorar a acessibilidade

Boas práticas básicas:

<!-- Alt em imagens -->
<img src="foto.jpg" alt="Descrição da foto">

<!-- Label em formulários -->
<label for="email">Email</label>
<input type="email" id="email">

5) Navegação por teclado

A navegação por teclado permite que o usuário use o site sem mouse, usando Tab, Enter e Setas.

button:focus {
  outline: 2px solid blue;
}

input:focus {
  outline: 2px solid blue;
}

Código acessível vs não acessível

❌ Código ruim

<!-- Sem label -->
<input placeholder="Email">

<!-- Sem alt -->
<img src="foto.jpg">

<!-- Tag sem sentido -->
<div>Navegação</div>

✅ Código acessível

<!-- Com label -->
<label for="email">Email</label>
<input type="email" id="email">

<!-- Com alt -->
<img src="foto.jpg" alt="Foto do produto">

<!-- Tag semântica -->
<nav>Navegação</nav>
