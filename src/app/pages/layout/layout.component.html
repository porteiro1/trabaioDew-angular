Flexbox e Grid

8) Diferença entre Flexbox e Grid

Flexbox organiza elementos em uma direção. Grid organiza elementos em linhas e colunas ao mesmo tempo.

Flexbox — uma direção

flex-direction

Define se os itens ficam em linha ou em coluna.

.container {
  display: flex;
  flex-direction: row;
}

.container {
  display: flex;
  flex-direction: column;

9) Quando usar Flexbox?

✔ Menus de navegação
✔ Botões lado a lado
✔ Centralizar um elemento na tela
✔ Barras com logo + links
✔ Formulários em coluna

Grid — linhas e colunas

grid-template-columns

Define quantas colunas o grid terá e qual o tamanho delas.

grid-template-columns: repeat(3, 1fr);
grid-template-columns: 200px 1fr 200px;
grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));

grid-template-rows

  display: grid;
  grid-template-columns: repeat(2, 1fr);

9) Quando usar Grid?

✔ Layout de página inteira
✔ Galerias de fotos
✔ Painéis com colunas e linhas
✔ Sistemas de cards complexos

Resumo: Flexbox vs Grid

Flexbox

display: flex
flex-direction: row | column
justify-content: center | space-between
align-items: center | flex-start
gap: 1rem
flex-wrap: wrap

→ Melhor para 1 direção

Grid

display: grid
grid-template-columns: repeat(3, 1fr)
grid-template-rows: 100px 200px
gap: 1rem

→ Melhor para 2 direções
@todos os direitos reservados
