Setup inicial

Instalação do ambiente Angular

Para criar e executar projetos Angular, é necessário ter Node.js, npm e Angular CLI instalados. Depois disso, o projeto pode ser criado e rodado localmente.

$ npm install -g @angular/cli $ ng new meu-projeto $ cd meu-projeto $ ng serve --open

Node.js

Ambiente de execução JavaScript

O Node fornece o ambiente necessário para executar ferramentas JavaScript no computador. A versão recomendada é a LTS.

npm

Gerenciador de pacotes

O npm vem junto com o Node e gerencia dependências, scripts e ferramentas usadas no projeto, como o Angular CLI.
